Neden Css Kullanıyoruz?

CSS, farklı aygıtlar ve ekran boyutları için tasarım, düzen ve ekran varyasyonları da dahil olmak üzere web sayfalarınızın stillerini tanımlamak için kullanılır.

CSS Sözdizimi

CSS kural kümesi bir seçici ve bildirim bloğundan oluşur:

Örneğin h1 {color:blue;} şeklinde bir bildirimde bulunursak burada h1 seçici yani css i uygulamak istediğimiz elemandır, küme parantezi içindeki color property yani özelliktir, blue ise bu özelliğe atanan değer (value) dir.

CSS Eklemenin Üç Yolu Vardır

Harici CSS, Dahili CSS, Satır içi (İnline) CSS

Harici CSS projeye ayrı bir css dosyası ekleyip css kodlarımızı oraya yazıyoruz ve bu dosyasyı html sayfamıza sürükleyip bırakarak ekliyoruz.

Dahili CSS: Bu sayfada yukarıda head tagları arasında yazdığımız style kodları arasındaki css e dahili css denir.

Satır içi (İnline) CSS : Bu elementte kullandığımız style="background-color: brown; color: bisque;" kodları ile yazım şekline inline css denir. Bu şekilde de html elementlerine css yazabiliriz.

Style kullanarak html kontrolüne yazı tipi fontu verme

Html elementlerinde border ile çerçeve kullanabiliriz

Html elementlerinde renk ismi yerine renk kodlarını da kullanabiliriz

HTML öğesi için birden fazla stil belirtildiğinde hangi stil kullanılır?

1- Satır içi stili (İnline CSS) (html öğesi içinde h3 style="color: blue;" şeklinde yazarsak) 2- Dış ve iç stil sayfaları (baş bölümünde belirttiğim gibi css lerin sırasına göre) 3- Tarayıcı varsayılan (her tarayıcıcnın html elemanlarına uyguladığı varsayılan css özellikleri vardır)

CSS Arka Planlar

CSS arka plan özellikleri öğeler için arka plan efektlerini tanımlamada kullanılır.

arka plan rengi
arka plan resmi
arka plan tekrarı
arka plan eki
arka plan konumu

Css de arkaplan rengine opacity ile saydamlık, şeffaflık ekleyebiliriz
Gerekli kod : background-color: green; opacity: 0.3;
Opaklık için bir diğer yöntem de style da renk kodu olarak Rgba kullanmaktır
Örnek : style="background: rgba(0, 128, 0, 0.3)"
Sondaki değer alfa dır opaklığı bu değer ayarlar

CSS arka plan resmi

Css ile html elemanlarına arkaplan resimleri ekleyebiliriz

Bunun için yazmamız gereken kod şudur : background-image: url("arkaplan.gif");



CSS Arka Plan Tekrarı

background-repeat özelliği ile resim tekrarını yatayda, dikeyde veya tekrar etmeyecek şekilde ayarlayabiliriz

CSS arka plan pozisyonu

Özellik, arka plan resminin konumunu belirtmek için kullanılır.
Arkaplan resminin pozisyonunu ayarlamak için gerekli kod : background-position: right top; veya background-position: 25px 58px; pixellerle de ayarlayabiliriz.
Arka plan görüntüsünü sağ üst köşede konumlandırırız bu kodla.

CSS arka plan sabitleme

Özellik, arka plan görüntüsünün kaydırılması mı yoksa sabit mi olması gerektiğini belirtir (sayfanın geri kalanıyla birlikte kaydırmaz). Eğer sayfa aşağı kaydırıldığında resmin sabit kalmasını istersek şu kodu eklemeliyiz : background-attachment: fixed;
